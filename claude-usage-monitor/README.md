# Claude Usage Monitor

Claude의 로그인된 사용량 페이지를 주기적으로 확인해 Chrome 확장 프로그램 배지에 최대 사용률을 표시합니다. 설정한 기준을 넘으면 Chrome 알림도 보냅니다.

## 설치

1. ZIP 파일의 압축을 풉니다.
2. Chrome 주소창에 `chrome://extensions`를 입력합니다.
3. 오른쪽 위의 **개발자 모드**를 켭니다.
4. **압축해제된 확장 프로그램을 로드합니다**를 누릅니다.
5. 압축을 푼 `claude-usage-monitor` 폴더를 선택합니다.
6. Chrome에서 `claude.ai`에 로그인되어 있는지 확인합니다.
7. 확장 프로그램 아이콘을 누르고 **새로고침**을 한 번 실행합니다.

## 동작

- 기본 확인 주기: 30분
- 배지 숫자: 확인된 항목 중 가장 높은 사용률
- 배지 색: 초록(70% 미만), 주황(70% 이상), 빨강(90% 이상)
- 기본 알림 기준: 80%
- 자동 확인 시 비활성 Claude 탭이 잠시 열렸다가 자동으로 닫힙니다.
- 확인한 사용량과 설정은 Chrome 내부에만 저장됩니다.

## 참고

이 확장 프로그램은 Claude의 공식 사용량 API가 아니라 로그인된 사용량 페이지의 표시 내용을 읽습니다. Claude가 페이지 구조나 문구를 크게 변경하면 `parser.js` 수정이 필요할 수 있습니다.

## 제거

`chrome://extensions`에서 **Claude Usage Monitor**의 **삭제**를 누릅니다.
